# Front
This project was generated with Angular CLI version 14.2.13.

## Tools

- IntelliJ (IDE)
- Angular CLI

## Build
Open the front folder with your IDE (VsCode, IntelliJ...)
Run the command `npm install` to install node modules.
Run the application with `ng serve`
You can go to http://localhost:4200.

## Warning
Install the back-end before running the front-end.
